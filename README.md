# Online Movie Browsing System

I have developed a web application using Java Servlets and JSP, encompassing movie and TV series management, user account control, and payment processing. I allow administrators to add, edit, and delete movie and TV series details, provide user registration, login, and profile management features, and handle payment transactions for services or content access. Java Servlets manage the backend logic, including database operations and payment processing, while I use JSP to create user-friendly interfaces for these functionalities, resulting in a comprehensive platform for efficient and secure content management and user interactions.

## Features
- Movie Management

- TV Series Management

- User Management

- Payment Management
